# Smart Meeting Agent

Recruitment automation project using Python, Gmail API and Excel.

## Project Workflow

1. Create candidate database in Excel
2. Generate candidate emails automatically
3. Send applications to Gmail
4. Read Gmail inbox automatically
5. Extract candidate information
6. Calculate candidate score
7. Generate Excel reports

## Main Files

* main.py
* fake_mail_generator.py
* recruitment_agent.py
* create_token.py
* test_gmail.py

## Outputs

* candidate_results.xlsx
* gmail_candidates.xlsx
* gmail_summary.txt

## Technologies

* Python
* Gmail API
* Pandas
* OpenPyXL
* OAuth 2.0
